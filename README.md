# 🏥 Hospital Emergency Room Dashboard

## About This Project

This is a Power BI dashboard project that I created to analyze and visualize Hospital Emergency Room (ER) data. The main purpose of this project is to help understand patient flow, waiting times, admissions, referrals, and other important hospital statistics through interactive dashboards.

Instead of looking at raw data, this dashboard makes it easier to identify trends, compare monthly performance, and gain useful insights that can support better decision-making.

---

## Project Objectives

The main goals of this project are:

- Analyze emergency room patient data.
- Monitor patient waiting times.
- Track admitted and non-admitted patients.
- Understand department referral patterns.
- Analyze patient demographics like age and gender.
- Compare hospital performance across different months.
- Present important insights in a simple and interactive way.

---

## Dashboard Pages

### 📅 Monthly View

This page shows the hospital performance for a selected month.

It includes:

- Total Patients
- Average Wait Time
- Number of Admissions
- Number of Patients Not Admitted
- Department Referrals
- Patient Age Distribution
- Gender Distribution
- Daily Patient Trends

---

### 📊 Consolidated View

This page provides an overall summary of the complete dataset.

It contains:

- Overall Patient Count
- Admission Statistics
- Average Waiting Time
- Department-wise Referrals
- Overall Hospital Performance
- Interactive Charts and KPIs

---

### 👤 Patient Details

This page displays detailed information about each patient.

Some of the available details are:

- Patient ID
- Age
- Gender
- Race
- Visit Date
- Wait Time
- Admission Status
- Department Referral

Users can filter and search patient records easily.

---

### 💡 Key Takeaways

This page summarizes the important findings from the dashboard.

Some examples include:

- Peak patient visit periods
- Departments with the highest referrals
- Average waiting time
- Admission trends
- Patient demographic insights

## How to Use

1. Download or clone this repository.

```
git clone https://github.com/ShyamjithC123/Hospital-Emergency-Room-Dashboard.git
```

2. Open the `.pbit` or `.pbix` file using Power BI Desktop.

3. Connect it to the PostgreSQL database (or import the dataset).

4. Refresh the data.

5. Explore the dashboard using the available filters and navigation buttons.
